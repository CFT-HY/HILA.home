HILA (lattice in Finnish) is a C++ programming framework and suite of high-performance computing (HPC) lattice field theory simulations.

Purpose of HILA: to make writing computational field theory (CFT) applications straightforward and intuitive, while producing optimized executables for different (super) computing platforms.

Supported accelerations are parallelization with MPI, GP-GPU computing with CUDA or ROCm/HIP and AVX vectorization etc.

Write Once -- Run Anywhere
